  • CS0104 (Tipo ambiguo): Especifica namespace completo
  • [FromBody] en GET: Cambia a [FromRoute]
  • string para fechas: Cambia a DateOnly

CondominioApp es una aplicación web ASP.NET Core (.NET 9)
para gestionar condominios con validación robusta de inputs.

  • Validación de tipos de datos (texto, número, fecha)
  • Validación de campos requeridos
  • Validación de formatos (email, teléfono, documentos)

  • Respuestas estandarizadas
